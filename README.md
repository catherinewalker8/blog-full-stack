# Full Stack Pet Blog

This project is a full-stack pet blog application where users can create and share posts about animals and pets. The website allows users to write blog posts about different pets, organise them by category, and view posts created by others.

The aim of the project is to demonstrate how a front-end interface, a back-end server, and a database work together in a full stack application.

Users can create posts about topics such as cats, dogs, birds, or other animals, making the website a simple platform for sharing pet-related stories, tips, and experiences.
